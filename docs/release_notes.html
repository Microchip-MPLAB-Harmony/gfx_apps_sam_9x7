New Features

New features added in this release are as follows:

  • Added Legato Quickstart WVGA configuration for the Raspberry Pi Touch Display MIPI DSI Display
  • Added Legato Quickstart WVGA configuration for the AC69T88A LVDS WVGA Display

    Note: Please contact your local Microchip Sales Office for access to the LVDS adapter and LVDS display.
  • Added Legato Quickstart 1280x400 configuration for the Waveshare 7.9inch DSI LCD Display
